🚀 VPS Deployment

Run these commands on your VPS to deploy CyberDiary:

# Install dependencies
pip install flask gunicorn
 
# Set a strong secret key
export SECRET_KEY="your-very-long-random-secret"
 
# Run with gunicorn (production)
gunicorn -w 4 -b 0.0.0.0:8000 app:app

💡 Use nginx as a reverse proxy on port 80/443 pointing to port 8000. The database file cyberdiary.db lives in the same folder as app.py.
